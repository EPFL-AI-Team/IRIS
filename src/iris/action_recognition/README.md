# Perception (Myriam)

Myriam's code for processing video/audio inputs.

Put your code here:
- `action_recognition/` - VideoMAE and action models
- `object_detection/` - Object detection models
- `ocr/` - OCR models
- `audio/` - Audio transcription
- `preprocessing/` - Video preprocessing utilities
- `datasets/` - Dataset loaders

Production code that Marcus can import and use.
