# VideoMAEv2 Finetuning - Modified Files

This directory contains only the files from the [official VideoMAEv2 repository](https://github.com/OpenGVLab/VideoMAEv2) that were modified to finetune the model on a custom dataset.

## Modified Files

- `build.py` - Dataset builder modifications
- `engine_for_finetuning.py` - Training engine adaptations
- `run_class_finetuning.py` - Main finetuning script customizations

These files have been adapted from the original VideoMAEv2 implementation to support the specific requirements of this custom dataset.
